Project Management System

A Frontend application developed to simulate the management system for academic projects, to be used by Faculty...

Details

  • User can Add new Students, Student Groups and Faculty to the system.
  • User can assign project to any of the groups as needed.
  • User can delete Students, Student Groups and Faculties and can delete the assigned project if required.
  • As the part of evaluation, User can Assign report ans presentation marks to the groups and Final marks will be calculated automatically.
  • Finally, the user can view any of the information needed about Students, Groups, Faculties and Marks assigned.

Specifications

  • Programming Language : Java
  • Application type : GUI, Console
  • GUI developed using Java Swings

Crowdfunding Website

A Frontend website developed to simulate the working for Crowdfunding website with interlinked webpages...

Details

  • User can signup and login to the system.
  • Two homepages are provided for the user to spread awareness about Crowdfunding.
  • In the second homepage, links for different social medias are provided.
  • The official Government of India funding links are provided for user to fund to the projects initiated by Govt. of India.
  • As a part of this website, three forms have been provided which can be accessed by dropdown menu. User can fill the form, can print the filled form and can click on submit button provided.
  • Finally, the user can logout from website.

Specifications

  • Website developed using HTML, CSS, Javascript, Angular JS and Bootstrap.
  • Application type : Frontend

Hospital Data Management System

A website developed to manage the data associated with hospital, can be used to manage the data...

Details

  • After entering in the website, user can add new patients, doctors, rooms, nurses and department.
  • User can select any option provided in the navigation bar on homepage and will be redirected to the respective webpage.
  • User can get information about hospital on the 'Info' section given in which the patient list is provided along with some other info.
  • User can update patient information and can delete patient also.

Specifications

  • Frontend is developed using HTML, CSS, Javascript and Bootstrap.
  • Application type : Frontend and Backend
  • Database(DBMS) used : Oracle
  • Backend connected using PHP
