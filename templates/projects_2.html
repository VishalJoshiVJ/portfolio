Cricket Match Management System

An Application to manage the data associated with a cricket match and to manage the live matches...

Details

  • User can Schedule a match, delete a scheduled match, can view the scheduled matches.
  • User can add the result of scheduled matches and the winner will be calculated automatically.
  • User can view the result of the match where in all match data will be displayed.
  • If any live match is to be managed, then user can do so by updating toss, scores, wickets of both teams provided match is scheduled already, and winner will be decided automatically.
  • User can schedule the matches of tournament.

Specifications

  • Programming Language : Python
  • Application type : GUI
  • GUI developed using Python Tkinter
  • Database(DBMS) used : SQLite

Password Validator/ Checker

An Application developed in Python to check whether the given password has been hacked or leaked before by anyone...

Details

  • This is a Console based application
  • User has to input the password to be checked to the console.
  • The password will be hashed using SHA1 Hashing algorithm.
  • Api service of pwnedpasswords.com will be used to know how many times the password is been hacked.
  • For the api service, the hashed version of password will be sent so that there will be no chances to leak that password.
  • Finally, the status of the password will be the output as how many times the password is hacked. And if it is never been, then respective message will be displayed.

Specifications

  • Application developed using python.
  • Application type : Console
  • Hashing Algorithm used : SHA1

Image Manipulator

An application developed to manipulate image like resize and convert the type of the image...

Details

  • User has to enter the directory in which the image is present on the machine.
  • User can type the desired dimensions of the image to be resized.
  • User can convert the image types eg. from JPEG to PNG, JPG to PNG and vice versa.
  • After filling all the required fields, the message will be displayed on the screen and the new image will be saved to the location as that of original image.

Specifications

  • Application developed using Python.
  • Application type : GUI
  • GUI developed using Python Tkinter.
